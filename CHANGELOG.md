# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.1] - 2025-11-10

### Added
- Initial release with monorepo structure
- Google AI (Gemini) middleware (@revenium/google-genai)
- Vertex AI middleware (@revenium/google-vertex)
- Shared core utilities (@revenium/google-core)
- Complete usage metering and tracking
- Custom metadata support for business context
- Streaming support for real-time responses
- Vector embeddings support
- Full TypeScript support with comprehensive types
- Fire-and-forget tracking (non-blocking)
- Circuit breaker pattern for reliability
- Configurable retry logic
- Debug logging support
- Comprehensive examples for all features
