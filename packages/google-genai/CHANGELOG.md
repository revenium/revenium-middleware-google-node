# Changelog

All notable changes to @revenium/google-genai will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.2] - 2025-11-11

### Fixed
- Fixed Vertex AI environment variable warnings when using Google AI (Gemini)
- Updated dependency: @revenium/google-core@^0.1.2

## [0.1.1] - 2025-11-10

### Added
- Initial release of Google AI (Gemini) middleware
- Support for chat completions with Gemini models
- Streaming responses support
- Vector embeddings support
- Custom metadata for business context
- Full TypeScript support
- Fire-and-forget metering (non-blocking)
- Comprehensive examples and documentation
